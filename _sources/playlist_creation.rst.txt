Создание плейлиста
==================

1. Вы можете создавать свои плейлисты для удобного просмотра любимого
   контента.
2. Для создания плейлиста выберите контент, который вы хотите добавить в
   плейлист.
3. Откройте меню плейлистов и выберите “Создать новый плейлист”.
4. Введите название плейлиста и нажмите “Создать”.
5. Готово! Теперь вы можете добавлять контент в свой плейлист.

`Вернуться обратно <index.html>`__
